初めまして！

あなたは学び続けている人ですか？

　私たちも学びを続けている人が集まっているギルドです。オーナー自身が「自分にとって学びやすい環境」が欲しいと思い作り始め、現在はメンバーと会話を交えながら「よりよい学びの環境」を日々作り上げています。そんな環境で、あなたも一緒に勉強しませんか？

実績

参加メンバー

49 人(7/4時点)

月間学習時間

1,608 時間(6/1〜6/30)

週間学習時間

370 時間(6/22〜6/28)

使い方

step.1

チャンネル「もくもく勉強机」に入室します

step.2

勉強し終えたたらチャンネル「もくもく勉強机」から退室します

step.3

チャンネル「勉強時間記録」から自分の勉強時間を確認してみましょう

Features

勉強時間記録

ボイスチャンネルへの参加と退室を勉強の開始と終了時間とし、勉強終了時に勉強時間を記録・出力します

週間勉強集計

毎週月曜日に先週の勉強時間を全メンバー分集計して結果を出力します

Github OpenSource

コードは全てGithubにて公開管理しています。興味あればご自由にご利用ください
