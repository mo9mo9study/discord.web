  • 勉強時間記録

    ボイスチャンネルへの参加と退室を勉強の開始と終了時間とし、勉強終了時に勉強時間を記録・出力します。

  • 週間勉強集計

    毎週月曜日に先週の勉強時間を全メンバー分集計して結果を出力します。

  • Github OpenSource

    コードは全てGithubにて公開管理しています。興味あればご自由にご利用ください。

  • ハッシュタグ

    ギルドの専用ハッシュタグです。
    「#もくもく勉強机」と「#もくもくオンライン勉強会」のハッシュタグを使うメンバーはギルドのメンバーの可能性が9割9分です。

  • 勉強道場

    勉強をする仲間が集まり、勉強する内容を宣言して開始し、終了とともに勉強内容の報告をします。
